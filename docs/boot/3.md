# 3. 32bit 보호 모드 전환

## 📖 알아야 할 내용
`cli` 모든 인터럽트를 중단하겠다는 명령어

`cr0` 0 -> 16비트 모드 1 -> 32비트 보호 모드

`gdb_start`
- NULL descrption 첫번째 메모리 공간에는 NULL

```dd 0x0000FFFF, 0x00CF9A00```

    1. 첫번째 메모리 주소
    - 두번째 맨 앞 `00` + 맨 뒤 `00` + 첫번째 `0000` = 0x00000000
    2. 구역의 크기
    - 두번째 중간 `F` + 첫번째 `FFFF` = 0xFFFFF
    3. 권한 
    - 두번째 중간 `9A` -> 코드 구역
    4. Flags
    - 두번째 중간 `C` -> 단위 4KB 뻥튀기
