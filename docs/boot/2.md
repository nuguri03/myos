# 2. 디스크에서 커널 읽어오기

## 📚 소스 코드
```asm
[ORG 0x7c00]
[BITS 16]

start:
    mov ax, 0x0800
    mov es, ax
    mov bx, 0x0000

    mov ah, 0x02    
    mov al, 1
    mov ch, 0
    mov cl, 2
    mov dh, 0
    mov dl, 0x80

    int 0x13

    jc disk_error

    mov si, msg
    call print
    jmp $

disk_error:
    mov si, err_msg
    call print
    jmp $

print:
    mov ah, 0x0e
.loop:
    lodsb
    cmp al, 0
    je .end
    int 0x10
    jmp .loop
.end:
    ret

msg db 'Successfully load kernel', 0
err_msg db 'Disk Read Failed', 0

times 510-($-$$) db 0
dw 0xAA55

times 1024-($-$$) db 0
```

## 📖 알아야 할 내용
### 1. cpu 레지스터의 종류
- **범용 레지스터**
1. EAX = 누산기 레지스터(rax, eax, ax, ah, al)
2. EBX = 베이스 레지스터
3. ECX = 카운트 레지스터
4. EDX = 데이터 레지스터

- **세그먼트 레지스터**
1. CS
2. DS
3. SS
4. ES
> 직접 쓰기 안됨, 간접 쓰기만 가능!

### 2. 16비트 cpu의 1MB 메모리 읽기
16비트 -> 2^16 = 64KB
\
20비트 -> 2^20 = 1MB
\
주소 = (세그먼트) * 16 + (오프셋) -> 20비트처럼 보이게 만듦

### 3. 하드디스크
```asm
    mov ah, 0x02    // 디스크 읽기 모드    
    mov al, 1       // 1개의 섹터
    mov ch, 0       // 0번째 실린더
    mov cl, 2       // 2번째 섹터를 읽음
    mov dh, 0       // 0번 헤드
    mov dl, 0x80    // 드라이브 번호(0 = 플로피 디스크, 0x80 = 하드디스크)
```
> 섹터 번호는 0-based-index가 아니다!
